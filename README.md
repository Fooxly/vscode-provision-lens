List all the TODOs and notes on top of each file for a quick overview.

## Extension Settings

You can set all the different types of todos/notes you want to set.

For example:
```
"todolens.types": [{
  "title": "üìù {0} Todos",
  "types": [
    "TODO",
    "FIXME"
  ]
},
{
  "title": "üìù {0} Notes",
  "types": [
    "NOTE"
  ]
}]
```
